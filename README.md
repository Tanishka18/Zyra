**Zyra**

Zyra is an Android application developed as a university software engineering project that explores continuous authentication using behavioral biometrics. Rather than relying solely on a one-time login, the application aims to continuously verify user identity by analyzing behavioral patterns such as typing dynamics, touch interactions, and contextual information.

The project is currently under active development, with core functionality implemented and additional features planned as part of ongoing research and development.

**Features**
1) Continuous authentication using behavioral biometrics
2) Behavioral monitoring through touch and typing interactions
3) Location-aware contextual analysis
4) Risk assessment dashboard
5) Timeline of authentication events
6) User profile and settings management
7) Privacy-focused on-device processing using TensorFlow Lite

**Tech Stack**
1) Kotlin
2) Android SDK
3) Gradle (Kotlin DSL)
4) TensorFlow Lite
5) Android Jetpack Libraries
6) Android DataStore
7) WorkManager
8) Glide
9) MPAndroidChart
